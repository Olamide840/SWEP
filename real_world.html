WannaCry Ransomware Attack

What Happened:

  • WannaCry spread via a vulnerability in Windows (EternalBlue).
  • It encrypted users files and demanded Bitcoin payments.

Impact:

  • Affected 200,000+ computers in 150+ countries.
  • Hospitals, transport, and corporations were disrupted.

Prevention:

  • Install security patches regularly.
  • Use antivirus software and offline backups.
  • Disable SMBv1 if unused.

Equifax Data Breach (2017)

What Happened:

  • Hackers exploited an Apache Struts vulnerability.
  • Sensitive data of 147 million Americans was exposed.

Impact:

  • Huge reputational damage and lawsuits.
  • Massive financial losses for Equifax.

Prevention:

  • Promptly apply security patches.
  • Regular vulnerability scanning and monitoring.

Yahoo Data Breaches (2013–2014)

What Happened:

  • Over 3 billion accounts were compromised due to weak security.
  • Stolen data included names, emails, passwords, and security questions.

Impact:

  • Loss of user trust and a lower acquisition price by Verizon.

Prevention:

  • Encrypt user data and enforce strong password policies.
  • Implement two-factor authentication.

Google Docs Phishing Scam (2017)

What Happened:

  • Attackers sent emails that appeared to be from Google Docs.
  • Users were tricked into clicking a link that requested access to their Google accounts.

Impact:

  • Millions of Gmail users were affected.
  • Accounts were hijacked, and personal data was exposed.

Prevention:

  • Always verify sender email addresses.
  • Don't click suspicious links or grant access to unknown apps.
  • Enable two-factor authentication on your accounts.

Stuxnet Worm (2010)

What Happened:

  • Stuxnet targeted industrial systems using Microsoft Windows.
  • It sabotaged Iran’s nuclear centrifuges by altering their speed.

Impact:

  • Destroyed nearly 1,000 centrifuges in Iran's nuclear program.
  • Revealed how malware can cause real-world physical damage.

Prevention:

  • Keep software and systems updated with security patches.
  • Isolate critical infrastructure networks from the internet.
  • Use advanced endpoint protection.

RockYou Password Leak (2009)

What Happened:

  • Hackers exploited a vulnerability in the RockYou database.
  • Over 32 million user accounts and passwords were stolen and stored in plain text.

Impact:

  • Exposed weak password habits of users.
  • Stolen passwords reused in other attacks (credential stuffing).

Prevention:

  • Always use strong, unique passwords for each service.
  • Never store passwords in plain text.
  • Use password managers and enable MFA (Multi-Factor Authentication).

Edward Snowden NSA Leak (2013)

What Happened:

  • Snowden, an NSA contractor, leaked classified documents to journalists.
  • Exposed global surveillance programs run by the U.S. and allies.

Impact:

  • Massive public debate on privacy and surveillance.
  • Damaged trust in government cybersecurity and handling of data.

Prevention:

  • Implement strict access control and user monitoring.
  • Apply least privilege principle (only access what’s necessary).
  • Use auditing tools to detect unusual data access patterns.

Marriott Data Breach (2014–2018)

What Happened:

  • Hackers accessed the Starwood guest reservation database via unsecured networks.
  • They harvested data by intercepting connections and exploiting vulnerabilities.

Impact:

  • 500 million guests had personal data stolen, including passport info.
  • Huge financial and reputational loss for Marriott.

Prevention:

  • Secure public and internal Wi-Fi networks with encryption.
  • Use VPNs when on public or untrusted networks.
  • Regularly monitor and audit network traffic.
